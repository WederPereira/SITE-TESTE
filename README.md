# 🌐 Página Web – Advogada

Este repositório contém o código-fonte de uma página web simples e elegante para apresentação profissional de uma advogada. O projeto foi desenvolvido utilizando **HTML** e **CSS**, com foco em clareza, responsividade e boa experiência do usuário.

---

## 📌 Sobre o Projeto

A proposta desta página é servir como um site institucional básico, apresentando informações essenciais como:

* Nome e área de atuação da advogada
* Breve biografia profissional
* Serviços oferecidos
* Informações de contato
* Layout limpo e profissional

Ideal para quem deseja uma presença online simples, funcional e de fácil manutenção.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** – Estrutura da página
* **CSS3** – Estilização e layout responsivo

---

---

## 🎨 Funcionalidades

* Layout responsivo para dispositivos móveis
* Seções organizadas (Sobre, Serviços, Contato)
* Design minimalista e profissional
* Código simples e fácil de editar
* Navegação fluida
